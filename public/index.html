Setup Guide (Webhook Mode)

Cách kết nối Zalo Bot qua Webhook

1

Tạo Bot trên Zalo

Mở Zalo → Tìm Zalo Bot Manager → Chọn Tạo bot. Lấy Bot Token từ tin nhắn Zalo gửi cho bạn.

2

Cấu hình Webhook trong Zalo Bot Creator

Vào Thiết lập chung của Bot, điền 2 ô:

Webhook URL: https://zalo-bot-webapp.vercel.app/api/webhook
Secret Token: my-zalo-webhook-secret

Secret Token phải khớp với biến WEBHOOK_SECRET trong Vercel Environment Variables. Mặc định là my-zalo-webhook-secret.

3

Setup Vercel KV (Redis) — 1 lần duy nhất

Vào Vercel Dashboard → Chọn project → Tab Storage → Create Database → Chọn KV (Upstash) → Bấm Create.

Vercel sẽ tự động tạo biến KV_REST_API_URL và KV_REST_API_TOKEN. Sau đó Redeploy lại project.

4

Chuyển sang Dashboard

Dán Bot Token vào tab Dashboard và bấm Kết nối. Nhắn tin trên Zalo → tin nhắn sẽ hiện ở đây!
