Quick Start Guide

Cách tạo và kết nối Zalo Bot

1

Mở Zalo Bot Manager

Mở ứng dụng Zalo → Tìm kiếm OA Zalo Bot Manager → Nhấn vào menu chat và chọn Tạo bot.

2

Đặt tên Bot

Tên bot bắt buộc bắt đầu bằng "Bot" (VD: Bot AI Support). Sau khi tạo xong, hệ thống gửi Bot Token qua tin nhắn Zalo.

123456789:abc-xyz123 Ví dụ Token
3

Xóa Webhook (nếu có)

App này sử dụng chế độ Polling (getUpdates). Nếu bot đã có webhook, cần xóa webhook trước. App sẽ tự động làm việc này khi bạn kết nối.

4

Chuyển sang Dashboard

Dán Token vào tab Dashboard để kết nối và bắt đầu nhận/gửi tin nhắn real-time.

Về chế độ Polling

App này dùng API getUpdates (Long Polling) qua Proxy serverless. Phương thức này loại trừ lẫn nhau với Webhook — nếu bot đang có webhook, getUpdates sẽ bị Zalo chặn. App sẽ tự động gọi deleteWebhook khi kết nối.
